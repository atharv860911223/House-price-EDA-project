# House Prices Exploratory Data Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a housing dataset to understand property characteristics and identify factors that influence house prices.

The analysis focuses on data quality, feature distributions, missing values, correlations, and relationships between variables before building predictive models.

## Dataset

The dataset contains information about residential properties, including features such as:

- Lot Area
- Overall Quality
- Year Built
- Living Area
- Number of Rooms
- Garage Features
- Basement Features
- Sale Price

## Objectives

- Understand dataset structure
- Identify missing values
- Detect outliers
- Analyze feature distributions
- Explore relationships with sale price
- Generate insights useful for machine learning models

## Analysis Performed

- Data Inspection
- Missing Value Analysis
- Statistical Summary
- Feature Distribution Analysis
- Correlation Analysis
- Outlier Investigation
- Data Visualization

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Questions Explored

- Which features have the strongest relationship with house prices?
- Are there significant outliers in the dataset?
- How are property characteristics distributed?
- Which variables may be useful for future predictive modeling?
